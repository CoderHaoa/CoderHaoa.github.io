Git初始化-git init 疑惑
发表于 2020-04-22| 更新于 2020-05-20
阅读量:|评论数:

在使用命令git init后，在内部会生成.git文件夹。

倘若若出现==Reinitialized existing Git repository in xxx/xx/xxx/xx== 并且没有生成.git文件！

主要原因：隐藏了.git 文件

解决：当前文件夹→查看→显示和隐藏→勾选隐藏的文件！
