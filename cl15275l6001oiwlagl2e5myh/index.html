上传本地项目至gitee（for window）
发表于 2020-04-22| 更新于 2020-05-20
阅读量:|评论数:

1、首先要安装并配置Git；

2、 在本地待上传的文件夹，右键选择：Git Bash Here

3、执行：git init

4、 执行：git add .（add空一格加点）

5、执行：git commit -m "注释的内容" #提交到本地仓库

6、执行：gitee： git remote add origin 远程项目的Https地址

7、执行：git push -u origin master -f #-f强制上传

8、 弹出帐号密码框时：输入账号密码。

9、上传成功，可在gitee仓库中查看。
